# VertiCare 最终硬件接线表

本表对应当前稳定 demo 版本。所有 GND 必须共地，尤其是 ESP32-C6、SG90 外部电源和 MQ-135 分压电路。

## 传感器与执行器

| 模块 | 模块引脚 | ESP32-C6 引脚 | 供电建议 | 说明 |
| --- | --- | --- | --- | --- |
| DHT11 温湿度 | DATA | GPIO4 | 3.3V | 采集环境温度和空气湿度 |
| 光敏电阻模块 | AO | GPIO0 | 3.3V | ADC 越大通常越暗，Qt 显示为亮度百分比 |
| MH-RD 雨滴模块 | AO | GPIO1 | 3.3V | 主检测通道，按干燥基线判断是否有雨 |
| MH-RD 雨滴模块 | DO | GPIO5 | 3.3V | 可选辅助通道 |
| MQ-135 空气质量 | AO | GPIO2 | 5V | AO 必须经 10k/20k 分压后接 ESP32 |
| 火焰传感器 | DO | GPIO3 | 3.3V | 数字触发，触发电平在 `config.h` 配置 |
| SW-420 振动 | DO | GPIO6 | 3.3V | 使用中断捕获短脉冲，判断维护活动 |
| SW-520D 倾斜 | DO | GPIO7 | 3.3V | 结构倾斜告警 |
| SG90 舵机 | 信号线 | GPIO10 | 独立 5V 推荐 | 0 度关闭，90 度模拟灌溉开启 |
| TTP223 触摸 | OUT | GPIO12 | 3.3V | 短按切换模式，长按进入录卡 |

## RC522 RFID

| RC522 引脚 | ESP32-C6 引脚 | 说明 |
| --- | --- | --- |
| SDA/SS | GPIO18 | SPI 片选 |
| SCK | GPIO19 | SPI 时钟 |
| MOSI | GPIO20 | SPI 主出从入 |
| MISO | GPIO21 | SPI 主入从出 |
| RST | GPIO11 | 复位 |
| IRQ | 不接 | 当前 demo 不使用 |
| 3.3V | 3.3V | 不要接 5V |
| GND | GND | 与 ESP32 共地 |

## 注意事项

- ESP32-C6 的 ADC 引脚不能直接承受 5V，MQ-135 AO 必须分压。
- SG90 工作电流波动较大，建议独立 5V 供电，并与 ESP32-C6 共地。
- 雨滴模块请确认 AO 接 GPIO1，DO 接 GPIO5；接反会导致判断异常。
- SW-420 旋钮建议调到“静止熄灭、敲击微闪”的位置，再用程序窗口确认维护活动。
- RC522 不要接启动敏感引脚；当前 GPIO18/19/20/21/11 组合已经过调整，避免开机进入下载模式。
